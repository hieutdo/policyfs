PolicyFS

Policy-routed FUSE filesystem plus maintenance jobs for multi-disk storage.

Docs Releases Source

What

PolicyFS mounts multiple storage paths under a single mountpoint. Reads and writes are routed by rules, and directory listings can use an index to keep HDDs asleep.

How you run it

PolicyFS is meant to be operated via systemd: one long-running mount daemon, plus timer-driven maintenance jobs.

Quickstart

sudo dpkg -i ./pfs_<version>_amd64.deb
sudo edit /etc/pfs/pfs.yaml
sudo systemctl enable --now pfs@media.service
